Getting Started With Karma Jasmine and code coverage

Today, we will learn unit testing in JavaScript usking Karma, Jasmine and do code coverage. We write unit test to make sure each part of our code is working independently as per expectation.

I'm writing this for the beginner who want to setup the environment and start writing the unit test case.

Prerequisite

Must have NodeJS and npm installed
Install any editor of you choice

What we will be using

Jasmine - JavaScript framework for doing testing
Karma - Task runner which will execute jasmine test cases
Istanbul Code Coverage - Used for code coverage

Setting up Environment

First create a new folder for your code and then set up environment.
Now, will create package json file which has all the information about your web app.
Execute this command and follow the steps. 

npm start

First create a new folder for your code and then will install all the dependencies.
So, to install the dependencies will execute this command in command prompt. 

npm install jasmine jasmine-core karma karma-jasmine karma-chrome-launcher karma-coverage

All out js code will be in src folder and all test files will be in spec.
Now will do karma init to setup karma and follow the steps. 

npm karma init

This will create you karma.config file and add addition lines for code coverage

Now will write our JavaScript file and test file.

Now run this command to start karma and executes test cases. 

karma start

Now you can add all you source files inside src folder and start writing test cases inside spec folder. For complete srource code you can reffer to this git repo.
